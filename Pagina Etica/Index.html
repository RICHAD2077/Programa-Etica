Línea de Tiempo de la Evolución

1940s

Lenguaje Máquina

Código binario puro - la forma más básica de comunicación con el hardware.

; Ejemplo de código máquina (binario)
01001000 01100101 01101100 01101100 01101111
1950s

Lenguajes Ensamblador

Primera abstracción usando mnemotécnicos para representar instrucciones.

; Ejemplo de Assembly x86
MOV AX, 05h
MOV BX, 02h
ADD AX, BX
1957-1960

Primeros Lenguajes de Alto Nivel

FORTRAN (1957) para computación científica y COBOL (1959) para negocios.

C Ejemplo FORTRAN
PROGRAM HELLO
PRINT *, "Hello World"
END PROGRAM HELLO
1972

Nacimiento de C

Desarrollado por Dennis Ritchie en Bell Labs, se convirtió en el lenguaje de sistemas por excelencia.

// Ejemplo en C
#include <stdio.h>
int main() {
  printf("Hello, World!\n");
  return 0;
}
1995

Java y JavaScript

Java revoluciona con "write once, run anywhere". JavaScript nace para la web.

// Ejemplo Java
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}
2000s

Lenguajes Modernos

Python, C#, Ruby y otros enfocados en productividad y expresividad.

# Ejemplo Python
def hello_world():
  print("Hello, World!")

hello_world()
2010-Presente

Era de la Especialización

Go, Rust, Swift, Kotlin y TypeScript - cada uno optimizado para casos específicos.

// Ejemplo Rust
fn main() {
  println!("Hello, World!");
}
